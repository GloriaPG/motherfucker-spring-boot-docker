# motherfucker-spring-boot-docker

# Requirements
* [Docker](https://docs.docker.com/engine/installation/)

# Build image
``` bash
$ ./gradlew build buildDocker
```
# Docker RUN

``` bash
$ docker run -p 8080:8080 -t gloriapg/gloria-spring-boot-docker
```
